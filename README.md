## 使用方法

### `Dockerfile` 中

```dockerfile
FROM baoziyoo/base-images:php-fpm-8.1

ENV DEBIAN_FRONTEND noninteractive

RUN set -ex \
    # && sed -i 's/dl-cdn.alpinelinux.org/mirrors.aliyun.com/g' /etc/apk/repositories \
    && apk update \

WORKDIR /var/www
CMD ["php-fpm"]
EXPOSE 9000

```

### `run`
```
docker run -d baoziyoo/base-images:php-fpm-8.1
```
